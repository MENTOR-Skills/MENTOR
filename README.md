# MENTOR

**Mentoring Engine for Navigating Topics and Organizing Research**

MENTOR 是一套面向本科生的科研入门教练技能组（Agent Skills）。它帮助学生在陌生研究方向中建立基本认知、理解研究文献、整理本人已有的科研工作，并基于可核对的材料完成科研表达。

本项目**不以**代替学生完成科研或自动生成论文为目标，也**不追求**整夜自动实验并投稿。

## 能做什么

当前规划的五项核心能力：

1. **陌生领域入门路径** — 按学生背景给出学习顺序、知识缺口与第一个动手任务  
2. **专题文献全景与关系图** — 围绕具体主题整理主线、代表工作与关系，并给出精读建议  
3. **单篇论文深度阅读** — 读懂问题、方法、证据与局限；必要时提取复现条件  
4. **本地科研阶段总结** — 在授权范围内审计代码、日志与结果，形成可核对的阶段报告  
5. **基于研究材料的学术写作** — 将结构化证据写成阶段说明、组会材料等（先列主张与证据，再成稿）

# literature-landscape.md 模板

写成可读的专题全景说明，不是检索实验日志。篇幅按标准档自然展开；需要期刊向长综述时改用 `survey-writer`。

```markdown
# <主题>：文献全景说明

## 1. 主题与范围
一小节：主题、时间窗口、文献类型；来自 scope.md。勿粘贴检索式与漏斗表。

## 2. 主线 / 问题线程
列出 2–4 条主线（问题设定或方法族）。每条用段落说明共识与分歧，再点名少数代表工作（用库内 id 或作者年份指代）。

## 3. 代表工作与方法族
综合叙述演进关系；避免「A 说…B 说…」作为唯一结构。预印本标明 preprint。

## 4. 关系说明
说明论文间引用、方法继承、观点冲突等。
- 有文献内证据 → 可写为事实，并指向依据。
- 模型推断 → 必须写「（推断）」或等价标注。

## 5. 可见缺口
据**当前短名单**指出空白或薄弱区；明确「可能遗漏，非全域断言」。

## 6. 参考文献
仅列出 `references.json` 中出现的条目（作者、年份、标题、venue、DOI/arXiv/链接）。
```

# 材料索引（artifact-index）

> 时间窗：YYYY-MM-DD — YYYY-MM-DD（或「最近 N 天」）  
> 授权根路径：  
> - `path/one`  
> - `path/two`

## 已索引材料

| 路径 | 类型 | 一句话用途 | 是否已读 | 备注 |
|------|------|------------|----------|------|
| `relative/or/absolute/path` | code \| log \| result \| doc | … | 是 / 否 | |

## 跳过 / 未读

| 路径或模式 | 原因 |
|------------|------|
| （忽略规则命中或体积过大等） | |

## 说明

- 类型取值：`code`（脚本/配置）、`log`（日志）、`result`（指标/图表）、`doc`（README/笔记）。
- 「是否已读」指本轮 F4 是否打开并用于写报告；未读项不得支撑「已完成」主张。
